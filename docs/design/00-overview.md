# 设计规范总览

## 核心原则

> **把 Web 的"大、圆、影"换成桌面的"紧、方、线"**

14px 字号、8px 圆角、边框分割、Monaco 主题同步、72ch 阅读宽度——这些组合起来，就是 CiteForge 从"网页套壳"变成"原生学术工具"的分界线。

## 实施优先级

### P0（立即修复）
- [x] Monaco 主题同步
- [x] 字号统一（14px 正文 + 12px 标签）

### P1（下个迭代）

- [x] 自定义滚动条
